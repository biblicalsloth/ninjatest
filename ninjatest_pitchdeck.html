the problem

No opponent

Prep is a solo slog against a PDF. Zero social pressure, zero accountability — the #1 reason aspirants quietly stop practicing.

Speed untrained

NinjatestWhy Now04 / 16
the solution

A ranked 1v1 battle where every question is a move.

REALTIME

Both players see the same question at the same instant. Clock-offset sync makes the timer fair to the millisecond.

SPEED-GRADED

100-pt base plus grace points every 5s saved — ×2 in Quant & DILR. Right + fast beats right + slow.

SERVER-AUTH

Scoring, timing, and ELO run in security definer RPCs. Client only renders — impossible to cheat.

ZERO-SUM ELO

Win, number goes up; lose, it goes down. Margin and rating-gap set the size. No inflation, no trophies.

AI EXPLAIN

Every wrong answer gets a streaming Claude explanation post-match — saveable to profile, shareable via public link.

NinjatestTraction12 / 16
moat

Defensibility compounds with every match.

🌐
Network effects & liquidity

A ranked ladder is only fun when there are opponents at your ELO. The first platform to reach matchmaking liquidity in a given exam owns it — late entrants face empty lobbies.

🏅
ELO is switching cost

A rating and rank you grinded for months is non-portable identity capital. You don’t restart from zero on a clone — the same lock-in that keeps players on chess.com.

🛡️
Server-authoritative integrity

Scoring, timing & ELO in security definer RPCs with RLS-locked questions. A cheat-proof rated system is genuinely hard to build — and trust is the product.

🧠
Proprietary difficulty data

Every match yields per-question, per-ELO timing & accuracy data — a live difficulty & discrimination model no static mock-bank has. It powers adaptive matchmaking & content.

🤖
AI explanations = shareable growth loop

Every saved Claude explanation is a public page at ninjatest.app/explain/<id>. Wrong-answer cards shared on Telegram/Reddit are organic, SEO-indexed acquisition — a content moat competitors can’t easily replicate.

The flywheel: more players → tighter matchmaking → better data → smarter difficulty & pairing → AI explanations shared publicly → new players arrive via SEO + social → more players. Content companies can copy the questions; they can’t copy the liquidity, the rating graph, or the explanation corpus.
